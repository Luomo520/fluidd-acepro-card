# v0.1.4

- 修复仅启用 Moonraker `ace_status` 组件时 ACE Pro 页面和卡片入口被隐藏的问题。
- 增加多实例库存变量识别，兼容 `ace_inventory_0` 和 `ace_inventory_1`。
- 安装器可识别面板被官方 Fluidd 更新覆盖的状态，并提示重新安装。
- 安装前校验当前 Fluidd 版本与定制构建版本，避免部署不兼容文件。
- 安装后校验 ACE 路由构建文件及 Nginx 实际提供的入口文件，防止假成功。
- 部署时自动修正静态文件读取权限，避免 Nginx 因上传来源权限差异返回 HTTP 403。
- 补充 ACE Pro 页面标题的中英文翻译，避免浏览器标题显示原始 i18n 键名。
- 安装和更新流程改为直接使用 Git `clone` 与 `pull`，不再分发压缩安装包。

# v0.1.3

- 修复独立控制页面缺少 CSS、Vue 和配置资源时显示原始模板的问题。
- 修复料槽材料、温度和颜色保存后刷新丢失，保存命令改为传递 ACE 实例与料槽索引。
- 修复 Fluidd 卡片更新料槽时使用无效索引参数，统一通过工具编号写入驱动。
- 保存全部库存后重新读取 ACE 状态，确保界面显示实际持久化结果。
- 安装脚本自动链接独立控制页面所需的六个静态资源。
- 安装菜单顶部增加 Fluidd、ACEPRO 驱动和面板版本信息。
- 驱动检测合并到面板安装流程，并明确“独立控制界面翻译”及其还原选项。

# v0.1.2

- 修复 5 秒状态轮询覆盖尚未保存的材料、温度和颜色输入。
- 保存后等待 ACE API 返回一致数据，再恢复自动同步。
- 材料输入改为可搜索下拉框，提供常用材料并允许自定义输入。
- 颜色格式无效时禁用保存，避免意外保存为黑色。

- 压缩 ACE Pro 卡片的桌面端高度，减少面板、状态行和输入框间距。
- 缩小料槽卷盘图，并合并硬件状态与 SKU 信息行。

# v0.1.0

- 在 Fluidd 仪表盘和导航中加入原生 ACE Pro 控制模块。
- 使用 Kobra-S1/ACEPRO Moonraker API 获取设备、烘干和四个料槽状态。
- 支持装载、卸载、耗材编辑、烘干和无限续料控制。
- 加入中文字符菜单、驱动/API 前置检测、自动备份和失败回滚。
- 支持 ACE 独立控制页面中文化及一键还原。
- 提供完整中文安装教程和 Fluidd 源码覆盖层。
